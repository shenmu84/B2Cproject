## 快速开始

### 环境要求
- Node.js >= 12.0.0
- npm >= 6.0.0
- 现代浏览器（Chrome、Firefox、Safari、Edge等）

### 安装步骤
1. 克隆项目
```bash
git clone [项目地址]
```

2. 安装依赖
```bash
cd front_end_pc
npm install
```

3. 启动开发服务器
```bash
npm run serve
```

4. 构建生产版本
```bash
npm run build
```

front_end_pc/
